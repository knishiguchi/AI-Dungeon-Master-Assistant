V1 Features:

1. Natural language rules lookup
Problem - DMs constantly pause to search rules and need the results quickly
Features - Voice search capability, fuzzy search, version toggle (2014 vs 2024 rules), synonym handling 
Example prompts:
“How does grappling work?”
“What is the frightened condition?”
“Opportunity attack rules”

2. Stat Block access + smart adjustments
Problem - DMs need stat blocks fast and often rebalance/tweak encounters mid-fight
Features - Fast monster search with voice capability, Clean statblock view
Example prompts: 
“Scale this monster for 5 level-4 players”
“Make this creature more undead-themed”
“Give this goblin a fire ability”

3. Instant NPC Generator
Problem - Players talk to random people constantly
Features - Both one-click output or live prompted results
One click output example:
Name
Voice/personality
Goal
Secret
Quirk
Appearance

Live prompt example:
“Shopkeeper in desert city”

4. Magic Item Generator
Problem - Players will randomly loot an area and should be rewarded with a magic item
Features - Option to flavor an existing magic item, voice search capability, result displayed in clean card format
Example prompt:
"Give me an uncommon magic item for a human wizard that likes to play practical jokes"

V2 Features:
1. User configured settings - which D&D version to use, which specific books/campaigns, requires user management

2. NPC + PC saving - allows for creative prompting

3. Initiative tracking + HP tracking ? (Combat flow assistant)

4. Tavern/location generator


Important UX Details
1. Speed over beauty - minimize clicks
2. When possible - prebuilt buttons instead of input fields
